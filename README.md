## üß≠ Overview

The Heaven Platform is built on **Binance Smart Chain (BSC)** and introduces the HVN token (BEP-20 standard).  
It is designed to evolve through multiple phases:

1. **Foundation and Design**  
2. **Development and Deployment**  
3. **Launch and Initial Offering**  
4. **Sustainable Growth and Token Development**  
5. **DAO Governance, Staking, and Lending**

For detailed technical and economic information, please refer to the [Whitepaper](./docs/whitepaper.md).
